# Signal

Static site ready for deployment.

## Structure

- `index.html` : home page.
- `capital.html` : first published game, Capital.
- `index.html` : Signal home page, with logo and tile embedded directly.
- `games/capital/index.html` : compatibility redirect to `capital.html`.

## Quick deployment

Netlify: drag and drop this folder into "Deploy manually".

GitHub Pages: place these files at the root of a repository, then enable Pages on the main branch.

Vercel: publish the folder as a static site.
